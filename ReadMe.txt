- Programa en C para crear una base de datos de diversas cuencas hidrográficas.

El programa registra:

1. El nombre de las cuencas, y dentro de éstas, las presas
existentes.
2. El nivel de agua en % de cada presa y cada cuenca, y la fecha correspondiente
 a cada registro. Sólo almacena los 100 últimos registros de cada presa.
3. Imprime en un calendario hecho para tal efecto las variaciones (en %) de la 
cantidad de agua en la presa con respecto al registro anterior.
4. Imprime la cantidad y variación media del agua por cuenca, por presa, o en el
 total de las cuencas hidrográficas.

La siguiente versión añadirá:

1. Sin límite para almacenar los registros del caudal de agua de cada presa.
2. Interacción con el disco duro para almacenar, recuperar y modificar la información.

Esta base de datos es la tarea final que realicé para la asignatura "fundamentos de
programación", de la UNED, en el curso 2018/19.
